# PAL POCs

Proof-of-concept projects built on [PAL](https://github.com/quasientio/pal).

## Projects

| Project | Description |
|---------|-------------|
| [controller](controller/) | JavaFX app that throttles, pauses, and step-throughs remote interceptable peers via PAL intercept callbacks |

## Prerequisites

- Java 17+
- PAL installed and on `PATH` ([installation guide](https://quasientio.github.io/pal/getting-started/))
- Docker (for etcd — see [`infra/`](infra/))
